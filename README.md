## Purpose
ToyTopia is an online marketplace designed for kids' toys. It allows families to explore and purchase toys from local sellers, promoting small businesses and providing a fun, vibrant shopping experience for children and parents.

## Live URL
[https://kids-toy-store-9bd64.web.app/](#)

## Key Features
- User authentication with Firebase (sign up, login, and password reset)
- Browse and search for toys with detailed information
- Add toys to wishlist (stored in local storage)
- Interactive ratings and reviews for toys
- Interactive Google fonts inter,
- Responsive and vibrant UI using Tailwind CSS and DaisyUI
- SwiperJS carousel for showcasing featured toys
- Toast notifications for user actions (success/error messages)
- Loading spinners for asynchronous requests
- Clean, modern icons with React Icons and Lucide React
